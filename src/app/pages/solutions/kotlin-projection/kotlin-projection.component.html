Kotlin Projection Library

Unobtrusive, convention based object-object projection.

Kotlin  Kotlin-Library 
Repository Packages Releases

Converting objects is generally a manual and tedious job; for example, projecting the data from a data transfer object to a domain model object. In many cases this also leads to tight coupling of application layers.

The ONIXLabs Kotlin Projection Library provides unobtrusive, convention based object-object projection that respects the Kotlin type system and reduces the requirement to write manual mapping code. The best case scenario that can be achieved with the projection API is a zero-configuration, single line of code!
