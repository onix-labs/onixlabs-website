Kotlin is becoming increasingly popular, especially with application developers on the Java stack who want to 10x their productivity. This library lends a helping hand, providing several features to enhance the Kotlin ecosystem. Many of the features in this library are designed to support our other Kotlin offerings, but why keep that to ourselves?

Features

Collections

Provides several useful extension functions over standard Kotlin collections; for example being able to perform ordered and unordered content checks over any collection that extends Iterable<T>.

Reflection

Adds several features to support Kotlin's reflection API; for example, being able to create types instances from a KClass, and obtain constructor, property, and type information.

Time

Extends Java's Time API with additional types and extensions.

Type Converters

Provides a pattern for implementing convention based, type-safe object-object conversion. Several type converters have already been implemented for many of Kotlin's standard types; for example, BigDecimal, BigInteger, Boolean, Byte, Char, Double, Float, Int, Long, Short, String, Type and UUID.

Extensions

The ONIXLabs Kotlin Core Library is supported by a plethora of extensions which wrap common tasks into easily callable functions, and massively extend the core API, allowing applications to become more robust and succint through code reuse. Extensions exist to enable developers to work better with Kotlin's standard types.
